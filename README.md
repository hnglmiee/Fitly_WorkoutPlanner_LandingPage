## üöÄ Getting Started

### Prerequisites

- Node.js 18+ and npm/yarn

### Installation

```bash
# Clone repository
git clone https://github.com/hnglmiee/Fitly-Landing-Page.git
cd Fitly-Landing-Page

# Install dependencies
npm install
# or
yarn install

# Start development server
npm run dev
# or
yarn dev

# Open http://localhost:5173
```

## üîß Available Scripts

